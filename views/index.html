Einleitung

StupidGSO Api ist eine REST-basierte Schnittstelle zum Stundenplan des Georg-Simon-Ohm-Berufskollegs in Köln (GSO).

Der originale Stundenplan ist hier zu finden und eine Mobile-optimierte Version anhand dieser API hier

Diese Schnittstelle holt den originalen Stundenplan und dessen Bausteine von der GSO-Seite und Übersetzt diese. Alle Daten werden aus Performancegründen für eine variable Zeit zwischengespeichert.

Weeks

Wochenzahlen für die ein Stundenplan verfügbar ist.

Call

GET {{url}}/v1/weeks

Parameter

Name Type Required
none

Response

[
  "51",
  "52",
  "01"
]

Types

Kategorien der Elemente.

Call

GET {{url}}/v1/types

Parameter

Name Type Required
none

Response

[
  "classes",
  "teachers",
  "rooms"
]

Elements

Namen der Elemente, die ein Stundenplan besitzen.

Call

GET {{url}}/v1/elements/{type}

Parameter

Name Type Required
type string true

Response

[
  "AVM11",
  "AVM12",
  "AVM13",
  "AVM14",
  "AVM15",
  ...
]
